## maybe-v2-be

#### Run with Docker

1. Create .env and copy contents from .env.example
2. `docker compose up`

\*\*\*If there is any changes in code, need to remove and rebuild maybe-v2-be image with `docker compose up`

#### Local

1. Create .env and copy contents from .env.example
2. `npm install`
3. `npm run dev`
4. (build) `npm run build`
